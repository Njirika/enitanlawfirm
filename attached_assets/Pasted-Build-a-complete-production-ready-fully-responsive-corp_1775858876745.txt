Build a complete, production-ready, fully responsive corporate law firm website for Enitan Afolabi & Company, with both frontend and backend, using a modern stack that works well on Replit.
Project overview
Create a premium, elegant, modern corporate law firm website for Enitan Afolabi & Company, a Lagos-based law firm founded in 1996. The website must feel authoritative, refined, warm, and trustworthy. It should not look like a generic template. It should be inspired by top-tier corporate law firm websites.
Use the following color direction:
•	Primary: deep dark brown
•	Secondary: warm tan/light brown
•	Neutral: white
Use white generously for clarity and premium spacing. Use the brown and tan tones for accents, buttons, active states, cards, borders, and sections.
The website must be extremely mobile responsive, including:
•	all public pages
•	navigation/menu
•	forms
•	practice area sidebar/tabs
•	admin login
•	admin dashboard
•	admin tables/cards
•	blog management
•	career applications view
•	contact messages view
Tech stack
Choose a modern, maintainable stack suitable for Replit. Preferred stack:
•	Frontend: Next.js latest stable version with App Router, React, TypeScript, Tailwind CSS
•	Backend/API: Next.js server actions and/or route handlers
•	Database/Auth/Storage: Supabase
•	CMS-like admin: custom admin dashboard inside the app
•	Form validation: Zod + React Hook Form
•	UI: clean custom UI with accessible components
•	Deployment target: Replit deployment-ready
If needed, use:
•	Supabase Auth for admin authentication
•	Supabase Postgres for data
•	Supabase Storage for blog images and optional document uploads
•	Markdown or rich text blog editor
•	Framer Motion for subtle animations only
